🚀 快速开始

第一步：生成图标

用浏览器打开 scripts/generate-icons.html，下载三个图标文件，放到 public/icons/ 文件夹

第二步：重新构建

npm run build

第三步：加载到Chrome

访问 chrome://extensions/，开启开发者模式，加载 dist 文件夹

第四步：开始使用

访问 https://chat.deepseek.com，看到悬浮球即可使用！
