### 完成步骤

**1. 生成图标（2分钟）**
```
用浏览器打开：scripts/generate-icons.html
点击三个Download按钮
将文件移动到：public/icons/
```

**2. 重新构建（30秒）**
```bash
npm run build
```

**3. 加载到Chrome（1分钟）**
```
chrome://extensions/ 
→ 开发者模式
→ 加载已解压的扩展程序
→ 选择 dist 文件夹
```

**4. 开始使用！**
```
访问：https://chat.deepseek.com
看到悬浮球，点击即可使用！
```
