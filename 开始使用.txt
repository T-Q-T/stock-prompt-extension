---------------------------------------------------
第一步：生成图标（5分钟）
---------------------------------------------------

方法一：使用自带工具（最简单）
1. 用浏览器打开这个文件：
   scripts/generate-icons.html

2. 你会看到三个图标预览
   点击每个"Download XX"按钮

3. 把下载的三个文件移动到：
   public/icons/ 文件夹

文件名应该是：
- icon16.png
- icon48.png  
- icon128.png


方法二：使用在线工具
访问：https://www.favicon-generator.org/
上传你喜欢的图片，生成图标

---------------------------------------------------
第二步：重新构建（30秒）
---------------------------------------------------

在命令行运行：
npm run build

看到这样的成功信息：
✓ Copied icon16.png
✓ Copied icon48.png
✓ Copied icon128.png

---------------------------------------------------
第三步：加载到Chrome（1分钟）
---------------------------------------------------

1. 打开 Chrome 浏览器

2. 地址栏输入：chrome://extensions/

3. 打开右上角"开发者模式"开关

4. 点击"加载已解压的扩展程序"

5. 选择你的项目中的 dist 文件夹

6. 完成！

---------------------------------------------------
第四步：开始使用！
---------------------------------------------------

1. 访问：https://chat.deepseek.com

2. 你会看到右侧的紫色悬浮球

3. 点击悬浮球或按 Ctrl+M 打开侧边栏

4. 添加你的第一个 Prompt！

---------------------------------------------------
功能介绍
---------------------------------------------------

✨ 添加 Prompt
- 点击"Add New Prompt"按钮
- 输入标题和内容
- 点击"Add Prompt"保存

📝 管理 Prompt
- 悬停在卡片上显示操作按钮
- 📋 复制：复制内容到剪贴板
- ✏️  编辑：修改标题和内容
- 🗑️  删除：删除这个Prompt

🔍 搜索
- 在顶部搜索框输入关键词
- 实时过滤结果

⚙️ 设置
- 点击右上角齿轮图标
- 启用/禁用插件
- 添加更多网站域名

⌨️ 快捷键
- Ctrl+M：打开/关闭侧边栏

---------------------------------------------------
添加更多网站
---------------------------------------------------

默认只在 chat.deepseek.com 显示
想在其他网站使用：

1. 打开侧边栏
2. 点击齿轮图标（设置）
3. 输入新网站地址（例如：https://chatgpt.com）
4. 点击"+"添加
5. 点击"Save Settings"
6. 访问新网站，悬浮球就会出现！

---------------------------------------------------
常见问题
---------------------------------------------------

Q: 看不到悬浮球？
A: 检查是否在已启用的网站上，刷新页面试试

Q: 快捷键不工作？
A: 确保焦点在网页上（不在地址栏）

Q: 数据会丢失吗？
A: 不会！所有数据保存在浏览器本地

Q: 如何备份？
A: 数据存在 localStorage，可以手动导出
   （未来版本会添加导入/导出功能）
