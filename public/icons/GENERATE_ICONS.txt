如何生成图标 / How to Generate Icons
================================

方法1：使用图标生成器（推荐）
----------------------------
1. 在浏览器中打开：../scripts/generate-icons.html
2. 点击每个"Download"按钮下载图标
3. 将下载的文件放到这个文件夹（public/icons/）


方法2：在线工具
----------------------------
推荐网站：
- https://www.favicon-generator.org/
- https://www.canva.com/
- https://realfavicongenerator.net/

上传一个图片，生成不同尺寸的图标，然后重命名为：
- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)


方法3：使用AI生成
----------------------------
提示词示例：
"Create a modern icon for a browser extension called 'Prompt Stock'. 
The icon should feature a chat bubble or message symbol with a gradient 
from purple (#667eea) to violet (#764ba2). Minimalist design, 128x128px."

然后调整尺寸生成三个版本。


需要的文件
----------------------------
✅ icon16.png  - 16x16 像素
✅ icon48.png  - 48x48 像素
✅ icon128.png - 128x128 像素

都准备好后，运行：npm run build
